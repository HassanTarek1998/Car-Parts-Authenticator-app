# Car-Parts-Authenticator-app
Automotive OEM Authentication Suite A Python-powered web application. By integrating Airtable as a secure backend and using Regex pattern. Built with Streamlit for a seamless user experience and designed for rapid deployment.
About this Tool Counterfeit car parts are a safety risk. This app was built to give car enthusiasts and mechanics a first line of defense.
- Verified Database: We cross-reference your part numbers against known genuine sequences.
-Trusted Sources: If a part is flagged as suspicious, we provide links to authorized dealers to ensure your next purchase is safe.
